# NeoShare - 轻量级文件共享平台

NeoShare是一款基于Python Flask开发的轻量级文件共享平台，提供公共文件共享和用户私有文件管理功能，具有现代化的科技感UI设计。

## 功能特点

### 1. 公共文件共享

- 无需登录即可上传和下载公共文件
- 支持多种文件格式的预览

### 2. 用户管理系统

- 用户名密码登录
- 个人文件目录和公共文件目录
- 上传头像和设置昵称
- 用户封禁和启用功能

### 3. 文件管理

- 文件上传和下载
- 支持预览常见格式文件（PDF、文本文件、脚本文件等）
- 支持编辑文本文件和脚本文件
- 提供文件搜索功能

### 4. 管理员功能

- 用户列表管理
- 对普通用户进行封禁和启用

### 5. 现代化UI设计

- 科技感深色主题
- 粉紫色点缀效果
- 响应式设计，适配不同屏幕尺寸
- 流畅的动画过渡

## 技术栈

- **后端**: Python Flask
- **数据库**: SQLite
- **前端**: HTML5, CSS3, JavaScript
- **框架**: Flask, Flask-SQLAlchemy, Flask-Login
- **样式**: CSS Grid, Flexbox

## 安装和运行

### 1. 环境要求

- Python 3.7+
- pip 包管理器

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 启动应用

使用改进版的CSS Grid布局，解决了导航栏遮挡问题：

```bash
python new_app.py
```

访问地址：http://127.0.0.1:5001

## 使用说明

### 公共文件共享

1. 访问首页 http://127.0.0.1:5001
2. 点击"📁 选择文件"按钮选择要上传的文件
3. 点击"上传文件"按钮完成上传
4. 上传的文件将显示在公共文件列表中
5. 其他用户可以直接下载或预览公共文件

### 用户注册和登录

1. 点击导航栏的"注册"链接
2. 填写注册信息（用户名、邮箱、密码）
3. 注册成功后自动跳转到登录页面
4. 输入用户名和密码登录
5. 登录后可以看到个人文件目录和公共文件目录

### 个人文件管理

1. 登录后进入仪表盘页面
2. 可以上传文件到个人目录或公共目录
3. 可以对个人文件进行预览、编辑、下载和删除操作
4. 可以将文件设为公共或私有

### 文件预览和编辑

1. 点击文件的"预览"按钮查看文件内容
2. 对于文本文件和脚本文件，可以点击"编辑"按钮进行编辑
3. 编辑完成后点击"保存更改"按钮保存修改

### 文件搜索

1. 在仪表盘页面的搜索框中输入关键字
2. 点击"搜索"按钮
3. 系统将显示所有匹配的文件

### 用户设置

1. 登录后点击导航栏的"设置"链接
2. 可以设置昵称和上传头像
3. 点击"保存设置"按钮保存更改

### 管理员功能

1. 使用管理员账户登录
2. 点击导航栏的"管理员"链接进入管理员面板
3. 可以查看所有用户列表
4. 可以对普通用户进行封禁和启用操作

## 配置说明

### 主要配置项

在 `new_app.py` 文件中可以修改以下配置：

- `SECRET_KEY`: 应用密钥，用于加密会话数据
- `SQLALCHEMY_DATABASE_URI`: 数据库连接字符串
- `UPLOAD_FOLDER`: 文件上传存储目录
- `MAX_CONTENT_LENGTH`: 最大文件上传大小限制（默认100MB）
